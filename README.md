## Installation

Download from http://example.com/FIXME.

## Run by jar file
1. Prepare the `data.xlsx`. Change the sheet name as `Sheet0`
2. Run the command:
   ```
   $ java -jar pharmacy-0.1.0-standalone.jar data.xlsx
   ```
3. On the same directory, there will be a `sql.txt`

## How to run
1. Prepare the `data.xlsx`. Change the sheet name as `Sheet0`
2. Run the command:
   ```
   $ lein run data.xlsx
   ```
3. On the same directory, there will be a `sql.txt`
